Writr

Writr is blogging website for writers and readers alike. It is a space for you to talk about literally anything! Unlike most blog websites, Writr simplifies writing and reading articles by removing all the unnecessary features and it’s completely free of advertising, which means a clutter-free user interface, allowing writers to focus on the important bit... writing good content.

The challenge

With hundreds, if not thousands of blogging websites on the internet today, how does one pick which website to write their content for the public to read? There’s a lot to consider when making such a decision, ease of use is one of the important factors in consideration.

A big incentive behind a blogger’s choice is revenue, the ability to earn money through articles published, but that cannot happen without readers. For there to be readers, well, there needs to be something to read. The idea is to get writers hooked by making the process of going from idea to publishing incredibly easy!

A beautiful clutter-free user interface alone isn’t enough, how about a newsletter that you don’t have to pay for, having readers subscribe to your newsletter is a good way to remind content producers to keep making great content. How about connecting with other like minded, seasoned bloggers for future opportunities.

02

Chat application

The magic of the internet lies in connecting people all over the world. This is a classic messenger, chat web application.

The challenge

A chat application is meant to be real time, meaning users have to see things as they happen, a user has to see who in their contact list is online/active and who isn’t, they have to see messages immediately as they come in, provided they have internet connection and so on.

As usual, it starts with thinking about the user experience. I decided to go with a very simple yet attractive user interface that puts user communication at the center of it all, giving as much information to the user without cluttering the user interface and making the app seem complicated. The easier it is to use, the better!

03

Live location based weather app

This is a web app with live location tracking for accurate weather information and relevant information about traffic and future forecasts based on your device location. You’re not locked in only viewing weather information for the location you’re in, location search is doable as well.

The challenge

The task was to build an application that can access a device’s real time location and provide detailed weather information with all related weather news for that specific location.

My approach was to ask for user permission to access device location data while using the application, access to location data first means relevant weather information can be provided accurately. The user isn’t limited to viewing their location’s weather information though, they can still lookup detailed weather information for any desired location.
